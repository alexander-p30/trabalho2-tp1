void NumeroDeConta::setNumero ( string  numero)

Método setter para número de conta.

Parâmetros
numeroString no formato "XXXXXX-Y" composta somente por números (0 a 9), em que Y é o dígito verificador. O dígito verificador Y é calculado utilizando o seguinte algoritmo: Seja o número de conta ABCDEF-G, calculamos x = (A + B + C + D + E + F) % 10 e avaliamos o resultado. Se o resultado for maior que 0, G será 10 - x, caso for 0, G será 0.
Excepções
invalid_argumentLança exceção caso o formato não seja obedecido, ou o dígito verificador seja inválido.
